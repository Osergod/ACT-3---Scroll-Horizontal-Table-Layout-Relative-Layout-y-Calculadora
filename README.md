# ACT-3---Scroll-Horizontal-Table-Layout-Relative-Layout-y-Calculadora

Per a cada subactivitat, genera una nova Activity dins una mateixa App, accessibles des d'un menú principal amb buttons d'accès als diferents exercicis:

Exercici 11: Crea una pantalla amb 7 botons posicionats horitzontalmet, però per poder visualitzarlos tots fes servir un scroll horitzontal. Estableish un toggle que faci desaparèixer les lineas d’scroll.

 

Exercici 12: 

Crear la següent taula del dibuix en un pantalla fent servir primerament un TableLayout, després un gridLayout i finalment una combinació de LinearLayouts en cas de compresió del element prepara la taula perque es comprimeixi la columna central

 

Els elements que posicionarem dins la taula serán vistas amb colors de fons diferents



Exercici 13:

Crea el dibuix que veu a continuació tot fent servir un Relative Layout



Exercici 14:

Crea una pantalla que mostri una calculadora senzilla funcional per pantalla. Ha de tenir com a mínim els butons d'operadors "+", "-",  "x" i "/", amb el seu botó de = per tal d'executar operacions. 

Format d'entrega: pdf de documentació demostrativa dels apartats de l'activitat amb enllaç al Github y arxiu .apk a apartat Releases
